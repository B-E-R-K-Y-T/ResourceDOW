===[ FDA <=> AIFC converter v1.0 by jTommy

Общая информация
  Эта консольная программа конвертирует звуковые FDA файлы из игры
  "W40k: Dawn of War" и "W40k: Winter Assault" в AIFC файлы игры
  "Homeworld II". AIFC файлы можно сразу слушать в Winamp'е, с
  помощью плагина Relic In_AIFx (http://www.winamp.com/plugins/details.php?id=272)
  или декодировать в WAV.

Как пользоваться
  Копируете конвертер в каталог с FDA файлами. Запускаете в командной
  строке: "fda2aifc *.fda". Будут созданы файлы с расширением .aifc.
  Эти файлы слушаете в Winamp или декодируете в WAV: "decShell *.aifc".
  С помощью aifc2fda конвертера можно конвертировать AIFC файлы обратно
  в FDA формат.

Файлы в архиве
  fda2aifc.exe - Конвертер FDA => AIFC.
  aifc2fda.exe - Конвертер AIFC => FDA.
  dec.exe - Официальный декодер AIFC => WAV, от Relic Entertainment Inc.
  enc.exe - Официальный кодер WAV => AIFC, от Relic Entertainment Inc.
  decShell.exe - Консольная оболочка для декодера, поддерживает
                 файловые маски. Написана для удобства декодирования
                 большого количества файлов.
  encShell.exe - То же самое, только для кодера.
  fda2aifc_src.zip - Исходники на Дельфи.
